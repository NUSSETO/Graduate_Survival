# Graduate Student Survival - clicker game

A fast-paced web clicker game where you play as a graduate student trying to write 1000 papers to graduate.

## Features
- **Simple Gameplay**: One-button interaction ("Write Paper").
- **Passive Progression**: Energy regenerates automatically.
- **Upgrades**: Buy "Better Beans" or hire an "Undergrad Assistant" to speed up your progress.
- **Random Events**: Watch out for "Advisor Meetings" or "Lab Equipment Failures"!
- **Visuals**: A messy desk aesthetic with sticky notes and reports.

## logic
- **Energy**: Starts at 5/sec regen.
- **Papers**: Cost 2 Energy.
- **Goal**: Reach 1000 Papers to Graduate.

## How to Run
This is a static web application (HTML/CSS/JS).

1. Clone the repo.
2. Open `index.html` in your browser.
   - OR run a simple local server: `python3 -m http.server 8080`

## Technologies
- Vanilla HTML5
- CSS3 (Flexbox/Grid)
- Vanilla JavaScript (ES6+)

No dependencies required!
